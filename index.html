<a id="various2" href="#" data-video="http://video-js.zencoder.com/oceans-clip.mp4;http://video-js.zencoder.com/oceans-clip.ogg" data-width="640" data-height="264" data-h5video-poster="http://video-js.zencoder.com/oceans-clip.png" >Html5 video</a>

  • data-video(required):one or more urls,sperated by comma
  • data-width,data-height(required):pixel number representing the video's width and height
  • data-h5video-autoplay,data-h5video-poster,data-h5video-preload(optical):the coresponding attribute (poster,preload,autoplay) of the video tag

Flash fallback
If the <video> tag can not play the provided videos,it will fallback to flash. So include a video url which flash can play(for example, mp4 or flv file) in the data-video attribute.

Try the link above. If you use firefox or opera which not be capble of handling h.264 codec, flash fallback will be used.
